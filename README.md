# 🎧 Moodsound – Aesthetic Multi-Page Music Mood Web App  
### _JavaScript Mini Project (Practicals 1–11 Implemented)_

---

## 📌 Overview  
**Moodsound** is a multi-page, aesthetic, mood-based music web application inspired by Spotify.  
Users can add songs, browse moods, manage playlists, view recommendations, and enjoy an animated modal UI.  

The project is built entirely using **HTML, CSS, and Vanilla JavaScript** and implements **all JavaScript Lab Practicals (1–11)**.

# 🌟 Features

### 🎶 Music Playlist System  
- Add new songs with title, artist, mood & cover image  
- Remove songs  
- Search through playlist  
- Mood-based filtering  
- Auto-save playlist using **localStorage**  

### 🎨 Modern Aesthetic UI  
- Neon gradients & soft glow  
- Glassmorphism cards  
- Big bold fonts  
- Smooth animations & transitions  
- Modal popup with scale animation  

### 🎧 UI-Only Player  
- Displays “Now Playing”  
- Shows song metadata & cover  
- Works with mood filters  

### ⏱ Auto-Play Timer  
- Plays next song every X seconds  
- Uses JavaScript `setInterval()` (Practical 11)

### ⭐ Recommended Songs  
- Preloaded from JSON-like data (Practical 10)  
- Single-click “Add to Playlist”

### 🌐 Fully Multi-Page  
Pages included:
- `index.html` (Home)
- `playlist.html`
- `add.html`
- `recommended.html`
- `about.html`

| **11** | Timers | Auto-play highlight timer |
